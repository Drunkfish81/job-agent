# 🚀 智能招聘Agent V7.1

一个基于AI的智能招聘系统，具备简历分析、岗位匹配、多轮对话和用户偏好记忆功能。

## ✨ 特性

- 📄 **智能简历分析** - 自动分析简历质量、评分、改进建议
- 🎯 **精准岗位匹配** - 基于向量搜索的智能岗位推荐
- 💬 **多轮对话** - 与AI求职顾问实时交流
- 🧠 **偏好记忆** - AI会记住你的偏好（薪资、公司类型等）
- ⭐ **自动收藏** - 自动收藏推荐的最佳岗位
- 📊 **技能差距分析** - 分析当前技能与目标岗位的差距
